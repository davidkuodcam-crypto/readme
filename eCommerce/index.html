UNIT 1

在自己的電腦裡，蓋一座真實的網站

我們將使用 LocalWP 這套免費軟體，讓你的電腦瞬間變成一台強大的網站伺服器。

核心觀念釐清：為什麼要用 LocalWP？它跟 WordPress 差在哪？

很多同學會把 LocalWP (環境軟體) 和 WordPress (網站系統) 搞混，甚至疑惑為什麼不直接上網買主機。我們這樣想：

  • LocalWP 就像是買了一支「全新的智慧型手機」，它為 App 提供了運行的硬體環境 (伺服器、資料庫)。在正式花錢租用雲端主機前，這是我們免費測試、盡情弄壞也不用怕的「沙盒」。
  • WordPress 就像是你下載安裝到手機裡的那款「App 應用程式」（例如 LINE 或 IG）。它是用來建置與管理網站內容的核心大腦。
1

下載並安裝 LocalWP 軟體

  1. 請前往 LocalWP 官方網站：https://localwp.com/
  2. 點選右上角大大的 「Download」 按鈕。
  3. 選擇你目前使用的作業系統（Windows 或 Mac）。
  4. 填寫基本資料（可隨意填寫），點擊 GET IT NOW! 下載安裝檔並完成安裝。
防呆注意： 安裝過程中，如果 Windows 防火牆跳出警告詢問，請務必勾選所有選項並點選 「允許存取 (Allow Access)」。否則後續網站會無法連線！
💡 網速太慢備案： 若學校網路下載過慢，可向老師索取存有安裝檔的 USB 隨身碟直接安裝。
2

建立你的第一個網站與環境配置

  1. 打開 LocalWP 軟體，同意服務條款後，點擊畫面中間的綠色按鈕 「+ Create a new site」。
  2. 選擇預設的 「Create a new site」，然後點選 「Continue」。
  3. 在 「What's your site's name?」 輸入網站名稱。
    切記：網域與資料夾名稱的限制，絕對不要輸入中文！請使用「純英文或數字」（例如：WZU-Shop）。
  4. 環境選擇畫面 (Choose your environment) 請保持預設的 「Preferred」，點選 「Continue」。 （這代表軟體會自動幫你配好最適合 WordPress 運行的 PHP 與 MySQL 資料庫版本）
極度重要
3

設定 WordPress 後台「管理員鑰匙」

在系統自動安裝網站前，你要先設定好未來登入這個網站後台的「最高權限」帳號密碼。請務必嚴肅對待這一步！

  • WordPress Username： admin (為求上課方便，統一設定為 admin)
  • WordPress Password：
    請輸入您絕對不會忘記的密碼！

    ⚠️ 強烈建議立刻寫在筆記本或手機備忘錄上！在本地端環境忘記密碼是非常難救回來的。

  • WordPress Email： 輸入你常用的 Email 即可。

確認無誤後，點擊右下角的 「Add Site」。

接下來軟體會花 1~2 分鐘為您建置資料庫並下載最新的 WordPress 系統，請耐心等待進度條跑完。

4

登入後台、改為繁體中文與 SEO 設定

安裝完成後，點擊軟體畫面右上角的 「WP Admin」 按鈕，這會打開瀏覽器並進入 WordPress 登入頁面，請輸入剛剛設定的帳密。

  1. 設定語言與時區：登入後，在左側黑色的選單欄中點擊 「Settings」 (設定)。
    • 找到 「Site Language」 下拉式選單，往下滑找到並點選 「繁體中文」。
      💡 實務觀念：為什麼電商網站一定要設定繁體中文？
      不只是為了看懂後台！正確的語系宣告 (zh-Hant) 能避免瀏覽器誤判跳出「自動翻譯」干擾使用者、確保結帳表單（如發票載具）符合台灣格式，並大幅提升中文關鍵字的 SEO 搜尋排名。
    • 找到 「Timezone」，選擇 「UTC+8」 或城市列表中的 「Taipei」。
    • 滑到最底端，點擊藍色按鈕 「Save Changes」。此時介面應該會瞬間變成中文。
  2. 更新語言包 (新版 WP 常見問題)：有時候切換語言後，選單還是半英半中。
    👉 請前往左上角 「控制台 (Dashboard)」>「更新 (Updates)」，滑到最下方如果有看到「翻譯更新」按鈕，請點擊它並等待完成。
  3. 設定永久連結 (極度重要！)：前往左側選單 「設定」>「永久連結」，將永久連結結構改為勾選 「文章名稱 (Post name)」，然後滑到最下方點擊「儲存設定」。

    為什麼這步是 SEO (搜尋引擎優化) 的第一關？

    預設的網址可能是毫無意義的參數（如 ?p=123）或包含冗長的日期。改為「文章名稱」後，你的網址會變成乾淨的結構，例如：

    http://myfirstweb.local/about-us/

    Google 的爬蟲機器人非常依賴「網址列中的單字」來判斷這個網頁在講什麼。這種乾淨的語義化網址，對未來的網站排名起著決定性的基礎作用！

UNIT 2

魔法時刻：一鍵匯入精美版型

環境建置完畢！現在我們要用全球最受歡迎的 Astra 佈景主題，在 5 分鐘內把剛剛的「毛胚屋」變成充滿質感的商業旗艦店。

實作前必看：這四個核心工具在做什麼？

在開始瘋狂點擊滑鼠之前，我們先建立「網站架構」的宏觀思維。請把接下來的操作想像成**「裝潢一間商業實體店面」**：

1

Astra 佈景主題 (Theme)

👨‍🎨 建築設計師：決定網站的基礎骨架（天花板高度、選單位置、預設字體）。
※ 規則：一個網站同一時間只能「啟用一套」佈景主題。

2

Starter Templates 魔法外掛 (Plugin)

🚚 樣品屋搬家公司：這是一個龐大的範本圖書館外掛。它能將設計師做好的「全套裝潢、假圖片、排版」一鍵匯入到 Astra 的骨架中，讓你不用面對空白畫面發呆。

3

Elementor 頁面編輯器

🛠️ 萬能裝修工具箱：匯入版型後，我們要修改裡面的文字和圖片。Elementor 讓我們能像「做 PPT」一樣，在網頁前端用滑鼠拖曳、直覺地編輯內容。

4

WooCommerce 電商樞紐外掛

🛒 營業執照與收銀機：負責處理商品上架、庫存管理與購物車結帳。把它裝上去，展示間就會瞬間升級為「能產生金流的店面」。

整體流程結論：
請 「建築師 (Astra)」 帶著 「搬家公司 (Templates)」，並約定使用 「裝修工具 (Elementor)」，幫我們把一間帶有 「收銀機 (WooCommerce)」 的豪華樣品屋，蓋進電腦裡！
1

安裝並啟用 Astra 佈景主題

  1. 在 WordPress 後台左側選單，點擊 「外觀 (Appearance)」 -> 「佈景主題 (Themes)」。
  2. 點擊畫面上方的 「安裝佈景主題 (Add New Theme)」。
  3. 在右上角搜尋框輸入：Astra。
  4. 找到紫色的 'A' 圖示版型，滑鼠移過去點擊 「安裝 (Install)」。
  5. 安裝完成後，務必點擊「啟用 (Activate)」！

    最新版 Astra 防呆機制 (畫面跳轉處理)

    如果您點擊「啟用」後，畫面突然跳轉到一個滿版的「Welcome to Astra」精靈畫面（裡面可能會有影片或要您按 Let's go），請不要被牽著鼻子走！

    👉 請直接尋找畫面最右上角的「Exit Guided Setup ✕」或「回到控制台」的文字點擊關閉它，回到我們熟悉的 WordPress 左側黑底選單控制台。

2

啟動 Starter Templates 魔法外掛

  1. 回到控制台後，上方通常會出現一個大大的 Astra 歡迎橫幅。請直接點擊橫幅中紫色的 「Install Starter Templates」 或 「Start Building Now」 按鈕。 （點擊後系統會在背景自動為您安裝並啟用這個搬家外掛）
  2. （若沒看到橫幅） 沒關係！點擊左側 「外觀」 -> 「Astra」，在儀表板中找到 Starter Templates 區塊並點擊啟用。
  3. ⚠️ 關鍵分水嶺（千萬別點錯）： 進入外掛畫面後，系統會問你要用 AI 建置還是傳統範本建置。
    請務必點選畫面右側的 「Classic Starter Templates (Build with Templates)」。
    理由：AI 版本雖然新穎，但客製化程度低且容易生成錯誤版型。傳統範本庫才是業界設計師實戰的主流選擇！
  4. 接著系統會詢問你要使用哪款頁面編輯器，請點選藍粉色圖示的 「Elementor」。
必看！避雷指南
3

挑選版型與「避開 SureCart 陷阱」

現在你進入了超華麗的「樣品屋圖書館」。但這也是新手最容易踩坑的地方，請務必按照以下策略操作！

業界秘辛：為什麼我會被迫安裝 SureCart？

Astra 開發團隊近期推出了自家的電商系統「SureCart」來挑戰老大哥「WooCommerce」。為了推廣，他們在 Starter Templates 裡放入了大量**「硬性綁定 SureCart」**的購物車版型。

如果你選到這些版型，即使安裝過程中你勾選了 WooCommerce，系統還是會強制安裝 SureCart，導致後續選單與設定引導全亂，且 SureCart 進階功能皆需付費。

🛡️ 最安全的建置策略：從「企業版型」開始

  1. 為了確保一個100%乾淨的 WooCommerce 環境，我們「不要」在這邊直接挑選 eCommerce (購物車) 分類的版型。
  2. 請在畫面上方分類選擇 「Business (企業/商業)」 相關的版型（例如：Agency, Restaurant, Clinic 等）。
  3. 避開圖片右上角標有 PREMIUM 的付費版型。找到一個喜歡的免費版型，點擊它預覽。
  4. Customize (自訂視覺)： 系統會問你要不要上傳 Logo、修改顏色與字體。建議保持預設（尊重原設計師美感），直接點擊左下角的 「Skip & Continue」 和 「Continue」。
  5. 最後填寫資料畫面： 這是廠商收集名單的行銷畫面，完全不需要填寫！滑到底部，確認 "Import Customizer Settings", "Import Widgets", "Import Content" 都有打勾後，點擊最下方的藍色按鈕 「Submit and Build My Website」。
魔法正在發生！ (請勿關閉視窗)
系統正在雲端為你下載龐大的圖片、排版設定和所需的外掛。請給它 1~3 分鐘的時間。看到滿天灑下彩紙的「Congratulations!」畫面，代表樣品屋蓋好啦！
👉 點擊右上角「X」關閉精靈，回到 WordPress 控制台。
4

上線前的安檢：清理被塞入的推廣外掛

即便我們選了企業版型，Astra 還是會經常偷塞一些推廣工具。為了避免干擾後續 WooCommerce 的實作，我們要做最後的大掃除。

  1. 回到 WordPress 左側黑色選單，點擊 「外掛 (Plugins)」 -> 「已安裝外掛 (Installed Plugins)」。
  2. 檢查清單中是否有出現名為 「SureCart」、「SureForms」 或 「SureRank」 等不必要的外掛。 （註：若有看到「Spectra」，它是 Gutenberg 的排版增強外掛，可保留以免首頁跑版。）
  3. 【如果有看到這些 Sure 系列外掛】：
    • 先點擊該外掛下方的 「停用 (Deactivate)」。
    • 重新整理頁面後，點擊紅色的 「刪除 (Delete)」，將它徹底移除！
  4. 太棒了！你的網站現在是一個擁有絕美外觀、且乾淨無比的優質基礎，準備好在下個單元安裝真正的營業執照了。
UNIT 3

電商樞紐：WooCommerce 震撼上線

突破單純的形象展示！我們將親手為網站掛上「營業執照」，安裝全球市佔第一的開源電商系統，開啟商品上架、購物車與線上結帳的完整商業功能。

觀念擴充：為什麼業界都愛用 WooCommerce？

與 Shopify 或 Shopline 等「月租型」平台不同，WooCommerce 是一套 100% 免費且開源 的系統。只要安裝它，你的網站會瞬間長出「商店」、「購物車」與「結帳」三大核心頁面。

它的優勢在於「資料全權掌握在自己手裡」，不需要被平台抽成，且擴充性極強（可輕鬆串接台灣的綠界科技、藍新金流或是黑貓宅急便）。

1

取得營業執照 (手動安裝 WooCommerce)

因為我們在單元二刻意避開了充滿陷阱的購物車版型，現在我們要用最乾淨正規的方式手動安裝。

  1. 在 WordPress 左側選單，點擊 「外掛 (Plugins)」 -> 「安裝外掛 (Add New Plugin)」。
  2. 在右上角搜尋框輸入：WooCommerce。
  3. 找到「紫色對話框裡有 Woo 字樣」的圖示，點擊 「立即安裝」，等待幾秒完成後點擊 「啟用」。

WooCommerce 初始化設定精靈 (避坑指南)

啟用後，畫面通常會跳轉到 WooCommerce 的歡迎設定精靈（要求填寫商店地址等）。請按照以下原則處理：

  • 商店基本資料： 國家選「台灣」，地址隨意填寫（上課練習用）。
  • 拒絕額外推廣功能 (極重要)： 過程中如果系統問你要不要加裝 Jetpack、Mailpoet、Google Listings 等額外免費商業功能，請全部取消勾選或點擊「略過」！ 這些外掛非常吃資源，會讓你的本地端網站慢到當機。
  • 完成設定： 一路按繼續直到回到控制台。
設定完成！看一下左側黑單，是否多出了「WooCommerce (用來看訂單)」與「商品 (用來上架)」兩大樞紐？
2

新增你的第一項商品 (圖解版)

新手疑惑破解：為什麼畫面上有兩個「新增商品」？

剛裝好 WooCommerce 進入首頁時，正中央的大藍色按鈕是「新手引導捷徑」；而左側黑色選單的「商品 > 新增」是「常態功能入口」。兩者點進去的功能完全相同！建議熟悉後統一習慣點擊左側黑名單的操作。

上架商品就像發佈一篇 Facebook 貼文一樣直覺，但有幾個欄位新手最容易搞錯，請務必看清楚位置！

  1. 前往左側選單 「商品 (Products)」 -> 「新增 (Add New)」。
  2. A. 標題與長描述 (畫面最上方大框框) 輸入商品名稱。下方最大的編輯區塊是「商品長描述」，這裡通常放圖文並茂的詳細開箱文、規格表。
    （它在顧客前台會顯示在商品圖片的「正下方」）
  3. B. 商品資料與定價 (畫面中間區塊) 往下捲動找到「商品資料」面板。在「一般」分頁中輸入 「原價」 與 「特價」。
    💡 小提示：只要有輸入特價，商品圖片上就會自動出現吸睛的 "Sale" 標籤！
  4. C. 商品短描述 (畫面最底部的框框) 繼續往下滑，你會看到「商品短描述」。這裡建議寫 3~5 句商品最核心的賣點（例如：24H出貨、純棉材質）。
    （這非常重要！短描述在前台會顯示在「價格的正下方、加入購物車按鈕的上方」，是影響轉換率的黃金地段！）
  5. 最容易弄錯的地方
    D. 設定商品圖片 (畫面最右側的側邊欄) 請將視線移到畫面「最右側的面板」，往下滑找到 「商品圖片 (Product image)」 區塊，點擊來上傳主視覺圖。
    新手大忌：把圖插進文章裡啦！
    千萬不要用左上角的「新增媒體」把圖插進文字框裡！若要讓商品縮圖出現在首頁和商店目錄中，絕對要設定右下角的「商品圖片」，否則前台就會出現灰色的佔位符！

檢視：高轉換率商品頁的四要素

在業界，商品上架不僅僅是把資料填上去，更是行銷的第一步：

  • 精準的標題與分類 (Categories)： 除了好名字，記得在右側面板勾選商品分類（如：3C配件、男裝），讓顧客能透過目錄快速篩選，這對大型商店至關重要。
  • 定價策略 (心理學)： 善用原價與特價的錨點效應，讓顧客覺得「買到賺到」。
  • 高品質且規格統一的圖片： 商品主圖決定了顧客點擊的慾望。強烈建議統一裁切為 正方形 (1:1，如 800x800px)，並使用 TinyPNG 壓縮至 200KB 以下，確保網站載入極速。
  • 黃金版位 (短描述)： 這是顧客猶豫是否加入購物車前最後看到的一段話。寫出痛點、強調獨家優勢，不要只寫無聊的規格！
BONUS 1

課後加碼：展示你的專屬網站

既然網站是在「自己的電腦」裡，其他人要怎麼連進來看呢？LocalWP 內建了超棒的「Live Link」功能，一秒產生對外公開的臨時隧道網址！

啟用 Live Link (臨時隧道)

利用這個功能，你可以讓全世界都能暫時連進你電腦裡的測試網站，進行火力展示！

  1. 回到 LocalWP 軟體的主畫面。
  2. 看向視窗的最下方邊緣，你會找到一行標示著 「Live Link」 的文字。
  3. 點擊旁邊的 「Enable (啟用)」 按鈕。
  4. 等待系統配置後，原本的地方會出現一組隨機網址（例如 https://xxxx.loca.lt），點擊旁邊的圖示複製它貼給朋友即可。

朋友點開網址，卻跳出「要求授權」對話框？

這是 LocalWP 的第一道「技術門鎖」，防護外人隨意連入你的電腦。對話框的帳密並非你的 WP 登入帳密！
👉 解法： 帳號與密碼就在軟體底部 Live Link 區塊旁邊，Username 通常為系統亂碼（如 unwritten），請點擊旁邊的 Password [Copy] 按鈕 複製，連同網址一起給朋友輸入才能進站。

朋友輸入密碼進去了，但只看到「即將推出 (Coming Soon)」？

這是第二道「內容門鎖」。WooCommerce 預設會開啟保護模式，非管理員的訪客看不到任何商品。
👉 解法： 進 WordPress 後台 ➔ 點擊左側 WooCommerce ➔ 設定 ➔ 找到頂部的 Site visibility (網站能見度)，將其改為 「上線 (Live)」 並儲存即可。

Live Link 按鈕反灰不能按？

官方為防止濫用，請點擊 LocalWP 軟體左上角的「人像圖示」，註冊/登入一個免費帳號後，底下的按鈕就會解鎖了！

隧道特性提醒：
這個網址是依賴你的電腦硬體運作的。只要你關閉了 LocalWP 軟體、或是把電腦蓋上睡眠，這條隧道就會立刻中斷，網址也會失效喔！
BONUS 2

下課前必做：打包網站帶回家

網站建置在「學校電腦」，下課前務必把它「匯出成壓縮檔」帶走，回家才能無縫接軌繼續編輯！

1

在學校電腦：打包匯出 (Export)

  1. 回到 LocalWP 軟體主畫面，在左側清單找到你的網站。
  2. 對著網站名稱 「按滑鼠右鍵」 ➔ 點選 「Export (匯出)」。
  3. 彈出視窗保持預設，點擊 「Export Site」，選擇存檔到桌面。
  4. 系統會打包出一個 .zip 壓縮檔。
  5. 👉 務必將這個 .zip 檔案存進隨身碟或上傳到 Google 雲端！
2

在家裡電腦：無痛還原匯入 (Import)

  1. 家裡電腦需先安裝好 LocalWP 軟體並開啟。
  2. 把你帶回來的 .zip 壓縮檔，直接用滑鼠拖曳 (Drag & Drop) 到 LocalWP 的軟體視窗正中間放開。
  3. 點擊 「Continue」 ➔ 「Import Site」，等待進度條跑完即可在家繼續開工！
BONUS 3

進階觀念：如何讓網站正式搬上雲端？

LocalWP 只是展示間。若要正式對外營業接收真實訂單，您需要為網站租一個「24小時不打烊的店面」與「門牌」。

上線的兩大實體支柱

1. 虛擬主機 (Hosting)

這是您的「網路實體房產」。一台24小時連上網路的伺服器，負責存放網站的圖片、資料庫與程式碼。業界口碑推薦：Cloudways (效能首選) 或 Hostinger (新手高CP)。

2. 頂級網域 (Domain)

這是您的「專屬門牌」（如 www.mybrand.com.tw）。需向網域註冊商 (如 GoDaddy, Namecheap) 租用，每年約需支付 10~15 美金維持所有權。

如何將 Local 網站搬到雲端主機？

這不需要懂複雜的資料庫代碼，業界最推薦使用 「All-in-One WP Migration」 這款專業搬家外掛：

  1. 在您的 Local 網站： 安裝該外掛，點擊「匯出 (Export)」➔「檔案」，下載一個 .wpress 備份檔。
  2. 在您的雲端主機： 透過主機商後台安裝一個全新的 WordPress，登入後同樣安裝這款外掛。
  3. 無縫還原： 點擊「匯入 (Import)」，將 .wpress 檔案拖曳進去。系統會自動把所有資料蓋過去，並神奇地幫您把網址由 .local 轉換成正式的 .com！
電商必備的資安檢查 (SSL 憑證)：
搬家上線後，務必在主機後台一鍵開啟 SSL 憑證 (Let's Encrypt)。讓網址列顯示 https:// 與「安全鎖頭」，顧客才敢在您的網站刷卡購物！
