商務網站架設完整指南

從本地端環境建置、精美版型一鍵匯入，到 WooCommerce 購物車上線。跟著這份講義，帶你一步步打造專業級的商業旗艦網站。

UNIT 1

在自己的電腦裡，蓋一座真實的網站

我們將使用 LocalWP 這套免費軟體，讓你的電腦瞬間變成一台強大的網站伺服器。

核心觀念釐清：什麼是 LocalWP？

很多同學會把 LocalWP (環境軟體) 和 WordPress (網站系統) 搞混。我們這樣想：

  • LocalWP 就像是買了一支「全新的智慧型手機」，它為 App 提供了運行的硬體環境。
  • WordPress 就像是你下載安裝到手機裡的那款「App 應用程式」（例如 LINE 或 IG）。
1

下載並安裝 LocalWP 軟體

  1. 請前往 LocalWP 官方網站：https://localwp.com/
  2. 點選右上角大大的 「Download」 按鈕。
  3. 選擇你目前使用的作業系統（Windows 或 Mac）。
  4. 填寫基本資料，點擊下載並完成安裝。
注意： 安裝過程中，如果 Windows 防火牆跳出警告詢問，請務必點選 「允許存取 (Allow Access)」。
💡 網速太慢備案： 若學校網路下載過慢，可向老師索取存有安裝檔的 USB 隨身碟直接安裝。
2

建立你的第一個網站與命名

  1. 打開 LocalWP 軟體，點擊畫面中間的 「+ Create a new site」。
  2. 選擇預設的 「Create a new site」，然後點選 「Continue」。
  3. 在 「What's your site's name?」 輸入網站名稱。
    切記：絕對不要輸入中文！請使用英文或數字。
  4. 環境選擇畫面請保持預設的 「Preferred」，點選 「Continue」。
極度重要
3

設定 WordPress 後台「管理員鑰匙」

在系統自動安裝前，你要先設定好未來登入這個網站後台的帳號密碼。

  • WordPress Username (帳號)： 建議統一輸入 admin。
  • WordPress Password (密碼)： 強烈建議立刻寫在筆記本上！忘記了會進不去後台。
  • WordPress Email (信箱)： 輸入你常用的 Email。

確認無誤後，點擊 「Add Site」，耐心等待進度條跑完。

4

登入後台並改為「繁體中文」

安裝完成後，點擊右上角的 「WP Admin」 進入後台並輸入剛剛設定的帳密。

  1. 登入後，在左側黑色的選單欄中點擊 「Settings」。
  2. 找到 「Site Language」 下拉式選單，點選 「繁體中文」。
  3. 找到 「Timezone」，選擇 「UTC+8」 或 「Taipei」。
  4. 前往左側選單 「設定」>「永久連結」，結構改為勾選 「文章名稱 (Post name)」。
    為什麼這步超重要？（SEO 關鍵設定）
    預設的網址可能是一串毫無意義的數字（如 ?p=123）或包含冗長的日期。改為「文章名稱」後，你的網址會變成乾淨的結構，例如：
    http://myfirstweb.local/sample-post/
    這種乾淨的網址不僅讓訪客一看就懂，更是 Google 搜尋引擎最喜愛的格式，對未來的網站排名非常有幫助！
  5. 滑到最底端，點擊 「儲存設定」。
UNIT 2

魔法時刻：一鍵匯入精美版型

環境建置完畢！現在我們要用 Astra 佈景主題，在 5 分鐘內把剛剛的「毛胚屋」變成充滿質感的商業旗艦店。

實作前必看：我們接下來要做什麼？

在開始點擊滑鼠之前，我們先來認識接下來會遇到的「四大核心工具」。我們可以把它們想像成**「蓋一間商業店面」**的過程：

1

Astra (佈景主題 Theme)

👨‍🎨 建築設計師：決定網站的基礎骨架、整體配色和選單位置。一個網站一次只能套用「一個」佈景主題。

2

Starter Templates (魔法外掛)

🚚 樣品屋搬家公司：這是一個範本圖書館，裡面有幾百套設計師做好的漂亮網站，讓我們能「一鍵匯入」整套排版，不用從零開始畫。

3

Elementor (頁面編輯器)

🛠️ 萬能裝修工具箱：強大的排版工具。我們必須指定它，未來我們才能像「做 PPT」一樣，用滑鼠拖曳來修改網頁上的文字和圖片。

4

WooCommerce (電商外掛)

🛒 營業執照與收銀機：負責處理商品上架與購物車結帳。把它裝上去，這棟美麗的房子就會瞬間升級為「能賺錢的店面」。

整體流程結論：
我們接下來的任務， বিচারপতি 「建築師 (Astra)」 帶著 「搬家公司 (Templates)」，並約定好未來要用 「裝修工具 (Elementor)」，幫我們把一間自帶 「收銀機 (WooCommerce)」 的豪華樣品屋，直接蓋進我們的電腦裡！

常見疑惑：佈景主題 (Theme) 跟 範本 (Templates) 到底差在哪？

簡單來說，這是一個「空殼」與「內裝」的關係：

  • 佈景主題 (Astra)： 就像蓋好一棟「毛胚屋」。它決定了天花板高度、大門位置（網站骨架與預設字體），但裡面空無一物。
  • 範本 (Starter Templates)： 就像直接買一套「樣品屋的全套家具與裝潢」。它包含了排版好的假圖片、文字和色彩配置，直接塞進你的毛胚屋裡，讓你不用從零開始佈置。
1

安裝並啟用 Astra 佈景主題

  1. 在 WordPress 後台左側選單，點擊 「外觀」 -> 「佈景主題」。
  2. 點擊上方的 「安裝佈景主題」。
  3. 在右上角搜尋框輸入：Astra。
  4. 找到紫色的 'A' 圖示，點擊 「安裝」。
  5. 安裝完成後，務必點擊 「啟用」！
    突發狀況破解（防呆機制）：
    如果您點擊「啟用」後，畫面突然跳轉到一個滿版的「Welcome to Astra」精靈畫面（裡面有影片介紹），請不要跟著按下一步！
    👉 請直接點擊畫面最右上角的「Exit Guided Setup ✕」關閉它，就能回到我們熟悉的控制台。
參考資訊：Astra 官方網站 https://wpastra.com/
2

啟動 Starter Templates 魔法外掛

  1. 啟用 Astra 佈景主題後，上方會出現一個大大的歡迎橫幅（寫著 Your Website, Ready in Minutes）。
  2. 請直接點擊橫幅中紫色的 「Start Building Now (或顯示正在啟用...)」 按鈕。
    點擊這個按鈕，系統就會自動幫你把 Starter Templates 這套「樣品屋圖書館」外掛安裝起來！
  3. （若沒看到橫幅） 點擊左側 「外觀」 -> 「Astra」，找到入門範本區塊並啟用。
  4. ⚠️ 關鍵分水嶺（千萬別點錯）： 進入下一步後，系統會問你要用 AI 還是傳統範本建置。
    請務必點選畫面右側的 「Classic Starter Templates (Build with Templates)」。
    (切記：不要選左邊的 AI 功能，否則後續操作畫面會完全不同！)
  5. 接著系統會詢問頁面編輯器，請務必點選 「Elementor」。
Elementor 編輯器官方網站：https://elementor.com/
挑選時刻
3

挑選並匯入你的商務網站

  1. （若畫面右側跳出 AI Builder 廣告小視窗，請點擊「Maybe Later」關閉它。）
  2. 你現在進入了「樣品屋圖書館」。透過上方分類（如 Business 企業版型）尋找喜歡的版型。
  3. ⚠️ 重要提醒： 避開圖片右上角有標示 PREMIUM (付費) 的版型。
  4. 點擊喜歡的免費版型預覽。
    注意：若您選擇的是「購物網站」範本
    系統會多跳出一個畫面問您要用哪種電商平台 (Choose eCommerce Platform)。
    請務必點選下方的 「WooCommerce」 (本課程專用系統)，千萬別選它推薦的 SureCart！
    為什麼不選 SureCart？
    SureCart 是版型商近期主推的自家產品，雖有新穎介面，但許多進階功能需付費解鎖，且台灣中文教學資源極少。WooCommerce 則是全球市佔第一、100% 免費開源的業界標準。
    如果手滑選錯了怎麼辦？
    👉 解法一（推薦/最乾淨）：回到 LocalWP 軟體，對這個網站按右鍵選「Delete」刪除，重新花 3 分鐘建一個新網站（沙盒的好處就在這！）。
    👉 解法二（手動替換）：進入 WordPress 後台左側的「外掛」，找到 SureCart 點擊「停用」並「刪除」。接著點選「安裝外掛」，手動搜尋並安裝 WooCommerce。
  5. Customize (自訂視覺)： 系統會問你要不要換 Logo、顏色與字體。建議保持預設（保留原設計師的美感），直接點擊左下角的 「Continue」。
  6. Select features (選擇功能)： 系統列出即將安裝的附屬功能（如聯絡表單、SEO）。請保持預設打勾，並再次檢查畫面上的 eCommerce 是否為 WooCommerce（若顯示 SureCart，請務必按 Back 退回修改）。確認無誤後點擊 「Continue」。
  7. 最後一步（填寫基本資料）： 這是廠商收集電子報名單的行銷畫面，完全不需要填寫！請直接留白，並點擊 blue 按鈕下方的文字連結 「Skip & Start Building (跳過並開始建置)」 即可。
  8. 等待安裝與關閉通知： 如果安裝完成後被導向「Finish Setup (完成設定)」的英文核取清單畫面，請直接點擊右上角的「Dismiss Setup X」關閉它，回到熟悉的控制台。
魔法正在發生！
系統正在幫你下載圖片、文字和版型設計。請給它 1~3 分鐘的時間，期間絕對不要關閉視窗！
UNIT 3

電商樞紐：WooCommerce 上線

突破單純的形象展示！為你的網站安裝「營業執照」，開啟商品上架、購物車與線上結帳的完整商業功能。

觀念擴充：什麼是 WooCommerce？

WooCommerce 是一套免費且強大的「電子商務外掛」。只要安裝它，你的網站會瞬間長出「商店」、「購物車」與「結帳」頁面。它能完美融入 Astra 的裝潢風格，且支援實體商品 or 數位虛擬商品的販售。

1

取得營業執照 (安裝電商系統)

小提示：你可能已經完成了！
如果您在「第二單元」匯入的是購物網站的版型，魔法搬家公司通常會自動幫您安裝好 WooCommerce。
👉 檢查方式： 看一下左側黑色選單，如果有看到「WooCommerce」跟「商品」這兩個選項，代表您已經安裝成功，請直接跳過這一步，進入「步驟 2」！
  1. （若左側沒看到 WooCommerce，請跟著以下步驟安裝）
  2. 在 WordPress 左側選單，點擊 「外掛」 -> 「安裝外掛」。
  3. 在右上角搜尋框輸入：WooCommerce。
  4. 找到紫色對話框圖示的 WooCommerce，點擊 「立即安裝」，完成後點擊 「啟用」。
  5. 系統會跳出歡迎設定畫面（填寫商店地址、產業類別等）。請依照畫面引導填寫。(金流與物流設定可以先略過)
參考資訊：WooCommerce 官方網站 https://woocommerce.com/
設定完成後，你會發現左側選單多出了「WooCommerce (看訂單)」與「商品 (管理庫存)」兩大樞紐！
2

新增你的第一項商品

建立商品就像發佈一篇 Facebook 貼文一樣直覺簡單。

  1. 前往左側選單 「商品」 -> 「新增」。
  2. 標題與文案： 在最上方輸入精準的商品名稱，並在下方大區塊填寫詳細的商品介紹。
  3. 設定價格： 往下捲動找到「商品資料」區塊，輸入 「原價」（如果有打折，也可以輸入特價）。
  4. 設定封面圖片 (最重要！)： 請將視線移到畫面「最右側的面板」，往下滑找到 「商品圖片」 區塊，點擊「設定商品圖片」來上傳主視覺圖。
    新手必看陷阱：圖片上傳錯地方啦！
    很多同學會習慣把主圖直接插入中間最大的空白文字框裡。那是給客人點進去看「詳細長文案」用的。
    若要讓商品出現在首頁和商店列表中，絕對要設定右下角的「商品圖片 (封面照)」，否則前台就會出現灰色的缺圖(山水畫)圖案喔！
    業界標準尺寸：強烈建議裁切為「正方形 (1:1)」（例如 800x800 像素），且單張圖片容量務必壓縮在 200KB 以內。
    常見錯誤：403 Forbidden 上傳失敗怎麼辦？
    如果您上傳圖片時出現 403 Forbidden 的紅字錯誤，這通常是因為您下載到了被瀏覽器強制轉檔的 AVIF/WebP 格式（即使副檔名是 .jpg，但骨子裡不是）。系統掃描到格式不符，為了安全就會將其阻擋。
    👉 解決方案：請不要用「右鍵另存圖片」，改用「螢幕截圖」的方式處理（詳見下方說明）！

成功上架的四要素 (Checklist)

一個高轉換率的商品頁面，必須具備以下條件：

  • 精準的標題與分類： 讓顧客一眼知道這是什麼，並能透過分類目錄快速篩選。
  • 吸引人的定價策略： 善用「特價」欄位，系統會自動在前端產生醒目的 Sale 標籤。
  • 高品質且規格統一的圖片： 除了主圖，記得在下方新增「商品圖庫」展示多角度情境。(⚠️ 新手大際：千萬別直接上傳好幾 MB 的手機原圖，會導致網站跑不動！請務必統一裁切成正方形並壓縮)。
  • SEO 友善的描述： 撰寫包含關鍵字的短描述，幫助 Google 搜尋引擎更容易收錄。

解鎖隱藏門檻：註冊免費帳號以解鎖功能

老師提醒：在新版 LocalWP 中，如果您發現最下方的 「Live Link Enable」 按鈕是灰色且完全按不下去的，這並不是軟體故障，而是官方為了解決網路濫用免費隧道的規定：

👉 解決步驟：

  1. 看向 LocalWP 軟體畫面的「最左上角」，點擊那個綠色背景的「空白人像圖示」。
  2. 軟體會跳出網頁引導您註冊或登入免費的 Local 帳號（直接使用 Google 帳號連動最快）。
  3. 確認登入： 網頁若顯示「Welcome to your Local account! ... Live Links are now unlocked.」，且人像圖示變成了您的頭像，代表解鎖成功！
  4. 此時回到軟體，下方的「Enable」按鈕就會由灰轉亮，正式為您敞開大門。

💡 實作盲點破解：為什麼給了網址，同學卻看不到我的網站？

這是許多學生即便開了隧道，別人卻仍看不到網站的最主要「盲點」。
原因： 這是因為 WooCommerce 預設開啟了「即將推出 (Coming Soon)」模式！系統為了保護未完工的網站，會自動把所有未登入的訪客導向 WordPress 登入頁面。

🛠️ 快速解決方案 (進入後台設定)：

  • 方案 A：全面對外公開 (最推薦)
    前往 WordPress 後台 ➔ 「WooCommerce」 ➔ 「設定」 ➔ 「Site visibility」 分頁，將狀態改選為 「上線 (Live)」 並儲存。
  • 方案 B：使用專屬私人連結分享
    若您想保持「即將推出」狀態，請勾選 「使用私人連結分享你的網站」。系統會產生一串帶有金鑰的長網址（如 .../?woo-share=...），複製這串連結給同學，他們就能繞過限制看到您的作品了！

利用這個功能，你可以產生一條安全的「隧道」，讓全世界都能連進你電腦裡的測試網站，互相火力展示！

  1. 回到 LocalWP 軟體的主畫面。
  2. 看向視窗的最下方邊緣，你會找到一行標示著 「Live Link」 的文字。
  3. 點擊旁邊綠色（或剛解鎖）的 「Enable (啟用)」 按鈕。
  4. 系統讀取一下後，原本的地方會出現一組綠色的隨機網址（例如 https://xxxx.loca.lt），點擊旁邊的圖示複製它。
  5. 把這段網址貼到班上的群組或用手機點開，大家就能直接逛你的網站囉！
⚠️ 臨時網址的特性（非常重要）：
這個網址是「臨時隧道」。只要你關閉了 LocalWP 軟體、或是把電腦關機，這條隧道就會立刻中斷，網址也會失效。
BONUS 2

下課前必做：打包網站帶回家

因為網站是架設在「學校電腦的硬碟」裡，下課前務必把網站「匯出成壓縮檔」帶走，回家才能無縫接軌繼續編輯！

1

在學校電腦：打包匯出 (Export)

  1. 回到 LocalWP 軟體主畫面，在左側清單找到你的網站名稱（例如 MyFirstWeb）。
  2. 對著網站名稱 「按滑鼠右鍵」。
  3. 在跳出的選單中，點選 「Export (匯出)」。
  4. 畫面會跳出設定視窗，直接使用預設值，點擊右下角的藍色按鈕 「Export Site」。
  5. 選擇存檔位置（建議存在桌面）。系統會開始打包，最後產生一個 .zip 的壓縮檔。
  6. 👉 務必將這個 .zip 檔案存進你的 USB 隨身碟，或是上傳到 Google 雲端硬碟！
2

在家裡電腦：無痛還原匯入 (Import)

  1. 在家裡的電腦，一樣先去官網下載並安裝好 LocalWP 軟體。
  2. 打開 LocalWP，然後把你從學校帶回來的那個 .zip 壓縮檔，直接用滑鼠拖曳 (Drag & Drop) 到 LocalWP 的軟體視窗裡面放開。
  3. 系統會自動偵測這是一個網站備份檔，請依照畫面指示一路點擊 「Continue」 ➔ 「Import Site」。
  4. 等待系統跑完進度條。大功告成！你現在可以在家裡繼續修改你的網站了！
💡 小提示：
因為這個 .zip 檔案裡面包含了整個 WordPress 系統和你的圖片，檔案通常會滿大的（幾百 MB 左右），所以請確保隨身碟或雲端硬碟有足夠的空間喔！
